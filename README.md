# Vue Nice elements

```js
import NiceElements from'vue-nice-elements'
import "vue-nice-elements/dist/NiceElements.css";
Vue.use(NiceElements)
```


## Includes

- NiceActions
- NiceButton
- NiceColor
- NiceDate
- NiceDropdown
- NiceDropdownSimple
- NiceFile
- NiceFilter
- NiceInput
- NiceLoading
- NiceModal
- NiceNotifications
- NiceProgress
- NiceProgressCircle
- NiceSandbox
- NiceSearch
- NiceSidewalk
- NiceText
- NiceTextarea
- NiceToast
- NiceToggle
