# Vue Nice elements

Vue:
```js
import NiceElements from 'vue-nice-elements';
Vue.use(NiceElements);
```

CSS:
```html
<link rel="stylesheet" href="/node_modules/vue-nice-elements/dist/NiceElements.css">
```

## Includes

- NiceActions
- NiceButton
- NiceColor
- NiceDate
- NiceDropdown
- NiceDropdownSimple
- NiceFile
- NiceFilter
- NiceInput
- NiceLoading
- NiceModal
- NiceNotifications
- NiceProgress
- NiceProgressCircle
- NiceSandbox
- NiceSearch
- NiceSidewalk
- NiceText
- NiceTextarea
- NiceToast
- NiceToggle
